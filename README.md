# 🌍 The Roam Report

A modern and responsive travel landing page built to practice front-end development by translating a Figma design into code.

The main objective of this project is to improve HTML, CSS, and JavaScript skills while following responsive design principles and writing clean, maintainable code.

> **Note:** The UI design is based on a Figma template. This project was developed for educational and portfolio purposes.
